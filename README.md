# 📊 EscalaFácil

Sistema web desenvolvido para aplicação prática de conceitos de Matemática Discreta, utilizando Grafos e Matrizes de Adjacência na organização de equipes e geração automática de escalas.

## 🎯 Objetivo

O projeto tem como finalidade demonstrar como conceitos matemáticos podem ser utilizados para auxiliar a tomada de decisões em ambientes organizacionais, analisando compatibilidades entre colaboradores e gerando escalas automaticamente.

## 🌐 Acesse o Projeto

Clique no link abaixo para acessar o sistema:

👉 **[Acessar EscalaFácil](https://leandrosilvan.github.io/escala-facil/)**

## 🚀 Funcionalidades

- Cadastro de funcionários
- Definição de disponibilidade
- Matriz de Adjacência
- Visualização de Grafos
- Geração automática de escalas
- Histórico de escalas geradas
- Sistema de sugestões com envio por e-mail
- Interface responsiva para dispositivos móveis

## 🧮 Conceitos de Matemática Discreta

- Grafos
- Vértices
- Arestas
- Grau dos vértices
- Matriz de Adjacência
- Análise de compatibilidade

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage
- Font Awesome
- EmailJS

## 📱 Responsividade

O sistema possui menu mobile e adaptação para diferentes tamanhos de tela.
